3. SISTEMA DE COPIAS DE SEGURIDAD (BACKUP & CONTROL DE VERSIONES)
--------------------------------------------------------------------------------
Hemos configurado Git para gestionar las versiones de forma profesional. 

¿CÓMO RECUPERAR UNA VERSIÓN ANTERIOR SI ALGO NO TE GUSTA?
1. Ver el historial de commits (puntos de guardado):
   > git log --oneline
2. Deshacer todos los cambios no guardados y volver al último punto de guardado:
   > git reset --hard HEAD
3. Volver temporalmente a un commit específico para ver cómo era:
   > git checkout <codigo-de-commit>
4. Volver a la última versión de trabajo después de explorar:
   > git checkout main (o master)

*Adicionalmente, se pueden crear respaldos comprimidos de toda la carpeta.*
